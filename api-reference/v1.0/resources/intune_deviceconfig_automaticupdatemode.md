# <a name="automaticupdatemode-enum-type"></a>automaticUpdateMode Enumerationstyp

> **Hinweis:** Die Verwendung der Microsoft Graph-APIs zum Konfigurieren von Intune-Steuerelementen und -Richtlinien erfordert dennoch, dass der Intune-Dienst vom Kunden [ordnungsgemäß lizenziert](https://go.microsoft.com/fwlink/?linkid=839381) ist.

Mögliche Werte für den Modus für automatische Updates.
## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|userDefined|0|Benutzerdefiniert, Standardwert, ohne Vorsatz.|
|notifyDownload|1|Benachrichtigung bei Download.|
|autoInstallAtMaintenanceTime|2|Automatische Installation bei Wartungen.|
|autoInstallAndRebootAtMaintenanceTime|3|Automatische Installation und Neustart bei Wartungen.|
|autoInstallAndRebootAtScheduledTime|4|Automatische Installation und Neustart zum geplanten Zeitpunkt.|
|autoInstallAndRebootWithoutEndUserControl|5|Automatische Installation und Neustart ohne Steuerung durch den Endbenutzer|
