# <a name="androidrequiredpasswordtype-enum-type"></a>androidRequiredPasswordType Enumerationstyp

> **Hinweis:** Die Verwendung der Microsoft Graph-APIs zum Konfigurieren von Intune-Steuerelementen und -Richtlinien erfordert dennoch, dass der Intune-Dienst vom Kunden [ordnungsgemäß lizenziert](https://go.microsoft.com/fwlink/?linkid=839381) ist.

Von Android geforderter Kennworttyp.
## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|deviceDefault|0|Standardwert des Geräts, ohne Vorsatz.|
|alphabetic|1|Alphabetisches Kennwort erforderlich.|
|Alphanumerisch|2|Alphanumerisches Kennwort erforderlich.|
|alphanumericWithSymbols|3|Alphanumerisches Kennwort mit Symbolen erforderlich.|
|lowSecurityBiometric|4|Es wird ein Kennwort erfordert, das auf Biometrie mit niedriger Sicherheitsstufe basiert.|
|numeric|5|Numerisches Kennwort erforderlich.|
|numericComplex|6|Komplexes numerisches Kennwort erforderlich.|
|Beliebiger Wert|7|Ein Kennwort oder ein Muster ist erforderlich, und alle sind akzeptabel.|
