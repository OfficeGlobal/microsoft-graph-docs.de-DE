# <a name="booleancolumn-resource-type"></a>BooleanColumn-Ressourcentyp

Die **booleanColumn** einer [columnDefinition](columndefinition.md)-Ressource gibt an, dass die Spalte einen Boolean-Wert enthält.

Es folgt eine JSON-Darstellung einer **booleanColumn**-Ressource.

}

## <a name="properties"></a>Eigenschaften
