# <a name="microsoft-teams-device-usage-reports"></a>Microsoft Teams-Gerätenutzungsberichte

Verwenden Sie die Microsoft Teams-Gerätenutzungsberichte, um Informationen über die Gerätenutzung von Microsoft Teams in Ihrer Organisation zu erhalten.

## <a name="methods"></a>Methoden

| Methode                                   | Rückgabetyp | Beschreibung                              |
| :--------------------------------------- | :---------- | :--------------------------------------- |
| [Benutzerdetails abrufen](../api/reportroot-getteamsdeviceusageuserdetail.md) | Stream      | Abrufen von Details zur Microsoft Teams-Gerätenutzung nach Benutzer. |
| [Benutzeranzahl abrufen](../api/reportroot-getteamsdeviceusageusercounts.md) | Stream      | Abrufen der Anzahl der täglichen eindeutigen Benutzer nach Gerätetyp. |
| [Benutzeranzahl für Verteilung abrufen](../api/reportroot-getteamsdeviceusagedistributionusercounts.md) | Stream      | Abrufen der Anzahl der eindeutigen Benutzer nach Gerätetyp im ausgewählten Zeitraum. |
