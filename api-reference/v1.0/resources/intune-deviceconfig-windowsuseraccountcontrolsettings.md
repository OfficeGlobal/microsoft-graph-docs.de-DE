# <a name="windowsuseraccountcontrolsettings-enum-type"></a>windowsUserAccountControlSettings-Enumerationstyp

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Mögliche Werte für Einstellungen für die Windows-Benutzerkontensteuerung.

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|userDefined|0|Benutzerdefiniert, Standardwert, keine Absicht.|
|alwaysNotify|1|Immer benachrichtigen.|
|notifyOnAppChanges|2|Benachrichtigung bei App-Änderungen.|
|notifyOnAppChangesWithoutDimming|3|Benachrichtigung bei App-Änderungen ohne Dimmen des Desktops.|
|neverNotify|4|Nie benachrichtigen.|
