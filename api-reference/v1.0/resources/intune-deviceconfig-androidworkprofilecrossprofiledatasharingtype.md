# <a name="androidworkprofilecrossprofiledatasharingtype-enum-type"></a>androidWorkProfileCrossProfileDataSharingType-Enumerationstyp

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Profil übergreifendes datenfreigabetyp für Android-Profile.

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|deviceDefault|0|Geräte-Standardwert, keine Absicht.|
|Konditionen|1|Verhindern der Freigabe.|
|allowPersonalToWork|2|Zulassen der Datenfreigabe Anforderung aus dem persönlichen Profil an das Arbeitsprofil.|
|noRestrictions|3|Keine Einschränkungen für die Freigabe.|
