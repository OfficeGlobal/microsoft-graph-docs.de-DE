# <a name="vpptokensyncstatus-enum-type"></a>vppTokenSyncStatus-Enumerationstyp

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Möglicher Synchronisierungsstatus, der einem Apple Volume Purchase Program-Token zugeordnet ist.

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|Keine|0|Standardstatus.|
|inProgress|1|Letzte Synchronisierung wird ausgeführt.|
|abgeschlossen|2|Letzte Synchronisierung erfolgreich abgeschlossen.|
|failed|3|Fehler bei der letzten Synchronisierung.|
