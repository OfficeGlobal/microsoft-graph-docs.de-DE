Bezeichnungen zum Bewerten von TV-Inhalten in Kanada
## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|allAllowed|0|Standardwert, alle TV-Showinhalte zulassen|
|allBlocked|1|Keine TV-Showinhalte zulassen|
|Untergeordnetes Element|2|Die C-Klassifizierung ist für Kinder im Alter von 2 bis 7 Jahren geeignet|
|childrenAbove8|3|Die C8-Klassifizierung ist für Kinder ab 8 Jahren geeignet|
|allgemein|4|Die G-Klassifizierung ist für ein allgemeines Publikum geeignet|
|parentalGuidance|5|PG, Elternsperre (Parental Guidance)|
|agesAbove14|6|Die 14+-Klassifizierung ist für Betrachter ab 14 Jahren geeignet|
|agesAbove18|7|Die 18+-Klassifizierung ist für Betrachter ab 18 Jahren geeignet|
