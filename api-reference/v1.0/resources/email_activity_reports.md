# <a name="email-activity-reports"></a>E-Mail-Aktivitätsberichte

Mithilfe der E-Mail-Aktivitätsberichte erhalten Sie eine allgemeine Übersicht über den E-Mail-Verkehr innerhalb Ihrer Organisation. Sie können auch das Widget E-Mail-Aktivitäten nutzen, um sich über Trends und Details bezüglich der E-Mail-Aktivität pro Benutzer in Ihrer Organisation zu informieren.

> **Hinweis:** Ausführliche Informationen zu anderen Berichtsansichten und -namen finden Sie unter [Office 365-Berichte – E-Mail-Aktivität](https://support.office.com/client/Email-activity-1cbe2c00-ca65-4fb9-9663-1bbfa58ebe44).

## <a name="reports"></a>Berichte

| Function                                 | Rückgabetyp | Beschreibung                              |
| :--------------------------------------- | :---------- | :--------------------------------------- |
| [Benutzerdetails abrufen](../api/reportroot_getemailactivityuserdetail.md) | Stream      | Erhalten Sie detaillierte Informationen über die von Benutzern ausgeführten E-Mail-Aktivitäten. |
| [Aktivitätsanzahl abrufen](../api/reportroot_getemailactivitycounts.md) | Stream      | Ermöglicht es Ihnen, einen Überblick über die E-Mail-Aktivitäten (z. B. wie viele E-Mails versendet, gelesen und empfangen wurden) in Ihrer Organisation zu gewinnen. |
| [Benutzeranzahl abrufen](../api/reportroot_getemailactivityusercounts.md) | Stream      | Ermöglicht es Ihnen, sich anhand der Anzahl eindeutiger Benutzer, die E-Mail-Aktivitäten wie Senden, Lesen und Empfangen ausführen, über Trends zu informieren. |
