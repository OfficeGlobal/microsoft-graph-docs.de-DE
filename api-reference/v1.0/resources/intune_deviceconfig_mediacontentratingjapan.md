## <a name="properties"></a>Eigenschaften
|Eigenschaft|Typ|Beschreibung|
|:---|:---|:---|
|movieRating|[ratingJapanMoviesType](../resources/intune_deviceconfig_ratingjapanmoviestype.md)|Bewertung für Japan ausgewählter Filme. Mögliche Werte: `allAllowed`, `allBlocked`, `general`, `parentalGuidance`, `agesAbove15`, `agesAbove18`.|
|tvRating|[ratingJapanTelevisionType](../resources/intune_deviceconfig_ratingjapantelevisiontype.md)|Bewertung für Japan ausgewählter TV-Sendungen. Mögliche Werte sind:`allAllowed`, `allBlocked`, `explicitAllowed`.|
