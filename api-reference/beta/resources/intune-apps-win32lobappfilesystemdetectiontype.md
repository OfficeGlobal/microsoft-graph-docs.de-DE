# <a name="win32lobappfilesystemdetectiontype-enum-type"></a>win32LobAppFileSystemDetectionType-Enumerationstyp

> **Wichtig:** Microsoft Graph-APIs unter der/Beta-Version können geändert werden; die Produktion wird nicht unterstützt.

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Enthält alle unterstützten Dateisystem-Erkennungstypen.

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|notConfigured|0|Nicht konfiguriert.|
|existiert|1|Gibt an, ob die angegebene Datei oder den angegebenen Ordner vorhanden ist.|
|modifiedDate|2|Datum der letzten Änderung.|
|createdDate|3|Erstelldatum.|
|Version|4|Versions Werttyp.|
|sizeInMB|5|Größen Erkennungstyp.|
