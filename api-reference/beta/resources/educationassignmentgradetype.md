# <a name="educationassignmentgradetype-resource-type"></a>Ressourcentyp educationAssignmentGradeType

> **Wichtig:** Die APIs der /Beta-Version in Microsoft Graph befinden sich in der Vorschau und können Änderungen unterliegen. Die Verwendung dieser APIs in Produktionsanwendungen wird nicht unterstützt.

Stellt dar, wie eine Zuordnung bewertet wird. Innerhalb der **assignments.grading** -Eigenschaft wird verwendet.

Diese übergeordnete Klasse kann nicht direkt in der Zuweisung-Eigenschaft verwendet werden. Es ist vorhanden, um mehrere unterschiedliche Möglichkeiten zum Note Zuordnungen zu ermöglichen.


## <a name="properties"></a>Eigenschaften

Keine.
