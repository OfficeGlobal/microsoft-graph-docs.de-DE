<span data-ttu-id="1f143-108">Bezeichner für Objekte in Aufgaben sind Zeichenfolgenwerte Dienst generiert.</span><span class="sxs-lookup"><span data-stu-id="1f143-108">Identifiers for objects in Tasks are service-generated string values.</span></span> <span data-ttu-id="1f143-109">.</span><span class="sxs-lookup"><span data-stu-id="1f143-109"></span></span> <span data-ttu-id="1f143-110">Die Werte sind 28 Zeichen lang sein und Groß-/Kleinschreibung beachtet.</span><span class="sxs-lookup"><span data-stu-id="1f143-110">The values are 28 characters long and are case-sensitive.</span></span> <span data-ttu-id="1f143-111">Wenn als übergeben, wird der Dienst eine einfache Format Validierung des Bezeichners, vorgehen, wenn Format Überprüfung fehlschlägt, die Anrufer ungültige Anforderung (400) Fehler, der angibt, des Problems beantwortet werden.</span><span class="sxs-lookup"><span data-stu-id="1f143-111">When passed as in, the service will do a simple format validation of the identifier, if the format validation fails, the callers will receive a Bad Request (400) error response indicating this problem.</span></span> <span data-ttu-id="1f143-112">Diese Fehlermeldung gibt einen Fehler in der aufrufenden Anwendung, wie an:</span><span class="sxs-lookup"><span data-stu-id="1f143-112">Receiving this error indicates a bug in the calling application, such as:</span></span>

- <span data-ttu-id="1f143-p103">Die aufrufende Anwendung hat den Bezeichner als Zeichenfolge verarbeitet, bei der nicht zwischen Groß- und Kleinschreibung unterschieden wird. Identifiers in Tasks are case-sensitive.</span></span>
- <span data-ttu-id="1f143-p104">Die aufrufende Anwendung hat den Bezeichner abgeschnitten. Bezeichner in Aufgaben sind 28 Zeichen lang.</span><span class="sxs-lookup"><span data-stu-id="1f143-p104">The calling application truncated the identifier. Identifiers in Tasks are 28 characters long.</span></span>
- <span data-ttu-id="1f143-p105">Die aufrufende Anwendung hat versucht, einen Bezeichnerwert für ein Objekt in Aufgaben zu generieren. Vom Client generierte Bezeichner werden nicht akzeptiert. Alle Bezeichner werden vom Dienst bei Erstellung der Objekte generiert.</span><span class="sxs-lookup"><span data-stu-id="1f143-p105">The calling application attempted to generate an identifier value for an object in Tasks. <span data-ttu-id="1f143-p106">Diese Überprüfung ist **kein Sicherheitsfeature**.
