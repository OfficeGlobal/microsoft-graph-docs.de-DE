# <a name="geolocationcolumn-resource-type"></a>Ressourcentyp geolocationColumn

> **Wichtig:** Die APIs der /Beta-Version in Microsoft Graph befinden sich in der Vorschau und können Änderungen unterliegen. Die Verwendung dieser APIs in Produktionsanwendungen wird nicht unterstützt.

Die **GeolocationColumn** für eine Ressource [ColumnDefinition](columndefinition.md) gibt an, dass die Spalte ein GeoLocation-Objekt enthält.

Es folgt eine JSON-Darstellung einer **GeolocationColumn** Ressource.

}

## <a name="properties"></a>Eigenschaften

Die Ressource **GeolocationColumn** enthält keine Eigenschaften.
