# <a name="geolocationcolumn-resource-type"></a>Ressourcentyp geolocationColumn

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Die **GeolocationColumn** für eine Ressource [ColumnDefinition](columndefinition.md) gibt an, dass die Spalte ein GeoLocation-Objekt enthält.
