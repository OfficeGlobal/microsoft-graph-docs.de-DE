# <a name="vpptokenactionfailurereason-enum-type"></a>vppTokenActionFailureReason-Enumerationstyp

> **Wichtig:** Microsoft Graph-APIs unter der/Beta-Version können geändert werden; die Produktion wird nicht unterstützt.

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Mögliche Ursachen für ein Apple Volume Purchase Program Token Action-Fehlers.

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|Keine|0|Keine.|
|appleFailure|1|Beim Apple-Dienst ist ein Fehler aufgetreten.|
|internalError|2|Interner Fehler.|
|expiredVppToken|3|Es ist ein Fehler aufgetreten, da das Apple Volume Purchase Program-Token abgelaufen ist.|
|expiredApplePushNotificationCertificate|4|Es ist ein Fehler aufgetreten, da das Apple Volume Purchase Program Push Notification Certificate abgelaufen ist.|
