# <a name="educationassignmentrecipient-resource-type"></a>Ressourcentyp educationAssignmentRecipient

> **Wichtig:** Die APIs der /Beta-Version in Microsoft Graph befinden sich in der Vorschau und können Änderungen unterliegen. Die Verwendung dieser APIs in Produktionsanwendungen wird nicht unterstützt.

In der **assignments.assignTo** -Eigenschaft verwendet. Diese übergeordnete Klasse ist abstrakt und Sie können nicht die Eigenschaft in Zuordnungen auf diesen Wert festgelegt. Diese übergeordnete Klasse darstellt, die die Zuordnung zugewiesen werden soll, wenn die Zuordnung veröffentlicht wird.


Die [EducationAssignmentClassRecipient](educationassignmentclassrecipient.md) -Ressource wird eine Unterklasse der diese übergeordnete Klasse.
