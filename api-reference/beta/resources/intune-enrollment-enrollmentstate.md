> **Wichtig:** Microsoft Graph-APIs unter der/Beta-Version können geändert werden; die Produktion wird nicht unterstützt.

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Noch nicht dokumentiert

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|unknown|0|Geräte Registrierungsstatus ist unbekannt|
|registriert|1|Das Gerät ist registriert.|
|pendingReset|2|Registriert, aber es wird über das Registrierungsprofil registriert, und das registrierte Profil unterscheidet sich von dem zugewiesenen Profil.|
|failed|3|Nicht registriert, und es gibt einen Registrierungsdatensatz.|
|notContacted|4|Das Gerät wurde importiert, aber nicht registriert.|
|gesperrt|5|Das Gerät ist als Benutzerlos registriert, es wird jedoch verhindert, dass es zur Benutzerregistrierung wechselt, da die APP nicht installiert werden konnte.|
