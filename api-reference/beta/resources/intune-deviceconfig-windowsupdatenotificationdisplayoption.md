# <a name="windowsupdatenotificationdisplayoption-enum-type"></a>windowsUpdateNotificationDisplayOption-Enumerationstyp

> **Wichtig:** Microsoft Graph-APIs unter der/Beta-Version können geändert werden; die Produktion wird nicht unterstützt.

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Anzeigeoptionen für Windows Update-Benachrichtigungen

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|notConfigured|0|Nicht konfiguriert|
|defaultNotifications|1|Verwenden Sie die standardmäßigen Windows Update-Benachrichtigungen.|
|restartWarningsOnly|2|Deaktivieren aller Benachrichtigungen, ausgenommen Neustart Warnungen.|
|disableAllNotifications|3|Deaktivieren Sie alle Benachrichtigungen, einschließlich Neustart Warnungen.|
