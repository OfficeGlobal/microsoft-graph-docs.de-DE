Typ der Konten, die für Windows10SecureAssessment ConfigurationAccount zulässig sind.

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|azureADAccount|0|Gibt ein Azure AD-Konto im Format AzureAD\username@tenant.com an.|
|domainAccount|1|Gibt ein Domänenkonto im Format Domäne \ Benutzer oder user@domain.com an.|
|localAccount|2|Gibt ein lokales Konto im Format des Benutzernamens an.|
