# <a name="managedappflaggedreason-enum-type"></a>managedAppFlaggedReason-Enumerationstyp

> **Wichtig:** Microsoft Graph-APIs unter der/Beta-Version können geändert werden; die Produktion wird nicht unterstützt.

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Der Grund für die Kennzeichnung eines Benutzers

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|Keine|0|Kein Problem.|
|rootedDevice|1|Die APP-Registrierung läuft auf einem Rooted/Unlocked-Gerät.|
|androidBootloaderUnlocked|2|Die APP-Registrierung wird auf einem Android-Gerät gestartet, auf dem der Bootloader entriegelt ist.|
|androidFactoryRomModified|3|Die APP-Registrierung wird auf einem Android-Gerät ausgeführt, auf dem die Factory-ROM geändert wurde.|
