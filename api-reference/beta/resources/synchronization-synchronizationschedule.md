# <a name="synchronizationschedule-resource-type"></a>Ressourcentyp synchronizationSchedule

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Definiert den Zeitplan zum Ausführen einer [SynchronizationJob](synchronization-synchronizationjob.md)verwendet.

## <a name="properties"></a>Eigenschaften
| Eigenschaft     | Typ   |Beschreibung|
|:---------------|:--------|:----------|
|Ablauf|DateTimeOffset|Datum und Uhrzeit, wann dieser Auftrag abläuft. Der Timestamp-Typ stellt die Datums- und Uhrzeitinformationen mithilfe des ISO 8601-Formats dar und wird immer in UTC-Zeit angegeben. Mitternacht UTC-Zeit am 1. Januar 2014 würde z. B. wie folgt aussehen: `'2014-01-01T00:00:00Z'`.|
|Intervall|Dauer|Das Intervall zwischen Synchronisierung Iterationen.|
|state|String| Mögliche Werte: `Active`, `Disabled`.|
