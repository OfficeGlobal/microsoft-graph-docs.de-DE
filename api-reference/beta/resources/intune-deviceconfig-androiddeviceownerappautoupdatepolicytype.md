# <a name="androiddeviceownerappautoupdatepolicytype-enum-type"></a>androidDeviceOwnerAppAutoUpdatePolicyType-Enumerationstyp

> **Wichtig:** Microsoft Graph-APIs unter der/Beta-Version können geändert werden; die Produktion wird nicht unterstützt.

> **Hinweis:** Die Microsoft Graph-API für InTune benötigt eine [aktive INTUNE-Lizenz](https://go.microsoft.com/fwlink/?linkid=839381) für den Mandanten.

Android-Gerätebesitzer mögliche Werte für Status der APP-AutoUpdate-Richtlinie des Geräts.

## <a name="members"></a>Elemente
|Element|Wert|Beschreibung|
|:---|:---|:---|
|notConfigured|0|Nicht konfiguriert; Dieser Wert wird ignoriert.|
|userChoice|1|Der Benutzer kann AutoUpdates steuern.|
|nie|2|Apps werden nie automatisch aktualisiert.|
|wiFiOnly|3|Apps werden nur über WLAN automatisch aktualisiert.|
|immer|4|Apps werden jederzeit automatisch aktualisiert. Es können Daten erhoben werden.|
